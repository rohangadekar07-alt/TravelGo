# TravelGO

TravelGO is a modern travel booking and inquiry management website.

## Features
- Popular International and Indian destinations.
- Inquiry Management System.
- Confirmed Booking System with simulated payment gateway.
- Admin Dashboard with revenue statistics and growth charts.
- Integration with MongoDB Atlas.

## Tech Stack
- Frontend: HTML5, CSS3, Vanilla JavaScript, Chart.js
- Backend: Node.js, Express, Mongoose
- Database: MongoDB Atlas (Cloud)

## Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Create a `.env` file and add your `MONGODB_URI` and admin credentials.
4. Run `node server.js` to start the application.
